Labs

I've experiments to run...

Here you can find some small web applications I host on this server.

They were mainly done for exercising or experimenting, but they may be able to occupy you for a few minutes.

  • Avatar is a procedural geometric avatar generator. It was designed to provide guest profile pics on my blog. It features a basic request API and Disqus integration.
  • DynImg is a dynamic image linker. It basically links several pictures to one URL by randomly choosing one each time the URL is called.
  • NameGen is a simple online fantasy name generator based on phonemes concatenation.
  • Quotes is just a quotes app for fun.

...there is research to be done...
